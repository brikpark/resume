Music Synthesizer


The purpose of the system is to read music note and timing information from a txt file and translate this information to music that is played through a set of speakers.We chose to implement a free play mode that mimics an instrumental keyboard in the interface software. The project consists of a computer program written in C that interfaces with a set of purpose built circuits through a LabJack U3 DAQ.
